Cache Proxy Server

A lightweight caching proxy server built using Python and Sockets.

Enhance your app's performance by caching HTTP requests for faster loading times.

Learn more
Features
  • In-memory caching for fast retrievals
  • Reduces network load with local cache storage
  • Simple, minimalist design for learning and experimentation
How to Run

Use the following command to start the server on your chosen port: 

              python cache.py <PORT>

Replace <PORT> with the desired port number (e.g., 8080).

About the Project

This proxy server is designed to cache HTTP requests, improving speed and efficiency for applications by reducing repeated fetches from the server. Ideal for learning about networking and caching.
